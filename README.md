# Dumpster - Русификатор
Инструкция по установке. 
1. Скачайте все файлы на ваш ПК в любое удобное место. 
2. Откройте папку с аддоном которая расположена \World of Warcraft\_retail_\Interface\AddOns\Dumpster
3. Добавьте файл Dumpster-ruRU.lua
4. Откройте папку с аддоном которая расположена \World of Warcraft\_retail_\Interface\AddOns\Dumpster
5. Замените файл Dumpster.toc
